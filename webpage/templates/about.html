About

Objective

The goal of our web application is to help you remove haze from images.

Tech Stack

We used Python (Flask) for our API. Other technical knowledge which was required was that of HTML5, CSS, Python, Flask, JavaScript

User Flow

We built our core website on Flask(Python). The image is sent as multipart data from website as a POST request to /upload endpoint where the image is processes to remove haze and a high quality haze free image is sent back to user.
